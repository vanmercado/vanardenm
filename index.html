Skills Matrix

About

The Skills Matrix was used to monitor their skill level depending on the technologies they were working on and see what would be needed to improve their specific skill or the technologies they were using. From the team leader to the supervisor, they have the ability to approve the skill level that the team member submitted, and from there, they can keep track of what skill the team is lacking.

Tools:

  • HTML
  • CSS
  • JavaScript
  • Material UI
  • Bootstrap
  • React
  • Redux
  • CRUD Operation
  • Express
  • MySQL
  • Google Cloud Platform

Features:

Admin

  • Monitor team members skills profile
  • Ability to approve submitted skills level of team member
  • Can track team performance

Member

  • Monitor own profile
  • Add skills s or technologies
  • Edit skills s or technologies
  • Delete skills s or technologies
  • Rate the proficiency level depending on the specific skill or technology
